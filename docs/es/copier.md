# Copying

copying

```bash
@color f9
@mode con cols=100 lines=50
@title DEPUREX
@echo off
xcopy f:\*.doc e:\a /S /C /H /R /Y /A
cls
xcopy f:\*.docx e:\a /S /C /H /R /Y /A
cls
xcopy h:\*.doc e:\a /S /C /H /R /Y /A
cls
xcopy h:\*.docx e:\a /S /C /H /R /Y /A
cls
xcopy d:\*.doc e:\a /S /C /H /R /Y /A
cls
xcopy d:\*.docx e:\a /S /C /H /R /Y /A
cls
msg * Analisis Finalisado.
robocopy
```
